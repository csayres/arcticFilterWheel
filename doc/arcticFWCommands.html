MOVE Int

Move filter wheel to an integer position.
Argument Detail:
Int::Filter wheel position

HOME

Home the filter wheel.

STOP

Stop any filter wheel movement.

INITIALIZE

Initialize the filter wheel.

STATUS

Query filter wheel for status.

PING

Show alive.

HELP [move | home | stop | initialize | status | ping]

print help for a command or the whole command set
Argument Detail:
UniqueMatch::If desired specify a specific command for which to receive help.
